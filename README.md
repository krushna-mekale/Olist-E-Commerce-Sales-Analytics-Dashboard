# Olist E-Commerce Sales Analytics Dashboard

## Project Overview
This project analyzes Olist E-Commerce sales data using SQL, Python, and Power BI.

## Tools Used
- SQL (PostgreSQL)
- Python (Pandas, Jupyter Notebook)
- Power BI

## Project Workflow
1. Data Cleaning using Python and SQL
2. Data Transformation
3. KPI Creation
4. Dashboard Development

## Key KPIs
- Total Revenue
- Total Orders
- Average Order Value
- Repeat Customers
- Customer Satisfaction Score

## Dashboard Pages
### 1. Executive Overview
- Revenue
- Orders
- Customers
- Profit

### 2. Sales Analysis
- Revenue by Category
- Revenue by State
- Monthly Sales Trend

### 3. Customer Analysis
- Repeat Customers
- Customer Distribution
- Customer Ratings
